# Changelog Entries (English)

This directory will contain English changelog entry files.

To add a changelog entry, create a new `.mdx` file with proper frontmatter.

Example:
```
---
title: "Version 1.0.0 Released"
description: "Initial release of the application"
version: "1.0.0"
releaseDate: "2025-01-01"
type: "major"
categories: ["added"]
---

# Version 1.0.0

## Added
- Initial application features
- User authentication
- Basic dashboard

## Changed
- Improved user interface

## Fixed
- Various bug fixes
```
